# Acceptance Metrics Memory

Assembly success requires programmatic evidence: components, bodies, metadata,
occurrences, joints, interference-free status, and positive physical
properties. Real capture/export is disabled in 0.4.1.

Fail closed with V1.5 failure codes when evidence is missing: METADATA_MISSING,
JOINT_MISMATCH, INTERFERENCE_DETECTED, or PHYSICAL_PROPERTY_MISMATCH.
