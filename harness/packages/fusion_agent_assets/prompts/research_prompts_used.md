# Research Prompts Archive

These prompts are historical research inputs. They are not runtime policy and
must not override AGENTS.md, CAD Spec schemas, facade allowlists, verifier code,
or current user instructions.

## Deep Research 01 - Core Fusion Agent/Harness

```text
Conduct a deep, broad, evidence-based research study to validate and improve an
architecture for a dedicated Autodesk Fusion 360 AI agent built around a local
Fusion MCP Server.

Evaluate best practices from MCP architectures, CAD automation, agent
engineering, verification-driven workflows, Fusion API/Fusion MCP capabilities,
tool-use agents, planning-execution-verification loops, benchmark design,
regression testing, geometry validation, constraint-based modeling, parametric
CAD generation, and human-in-the-loop controls.

Compare direct tool-calling agents, planner/executor/verifier designs, state
machines, and workflow engines. Identify failure modes, metrics, benchmark
suites, validation strategies, logging/audit approaches, memory systems,
retrieval systems, and evaluation methodologies. Recommend a production-grade
standalone Fusion modeling harness and a prioritized implementation roadmap.
```

## Deep Research 02 - Memory and Fusion Capability Expansion

```text
Research a memory system and future toolbox roadmap for a Fusion 360 AI
agent/harness.

Axis 1: Memory and continuous learning
- Design a simple, robust memory system with global and project scopes.
- Let the harness learn from failures, successful workflows, repair loops,
  modeling decisions, and stable user preferences.
- Prevent recurring mistakes by retrieving relevant prior context during
  planning, execution, verification, and repair.
- Compare file-backed Markdown memory, vector stores, episodic memory,
  semantic memory, workflow memory, skill memory, and execution trace memory.
- Evaluate retrieval, ranking, storage, summarization, consolidation, pruning,
  and human review.

Axis 2: Future Fusion capability expansion
- Focus on individual, student, and small-scale usage.
- Exclude enterprise PLM/ERP and large-team workflow concerns.
- Prioritize components and assemblies, drawings, CAM preparation, simulation,
  mesh/scan workflows, and advanced surface/freeform modeling.
```
